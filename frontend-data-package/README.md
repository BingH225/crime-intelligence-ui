# Frontend Data Package (TiDB + Local)

This zip contains all data artifacts needed to run and demonstrate the frontend charts:

1. `tidb_api_exports/`
- Direct API dumps exported from backend connected to TiDB (`http://127.0.0.1:8002`).
- Includes all major GET endpoints used by frontend pages and one POST victim dashboard sample.

2. `local_data/data_cache/`
- Existing local cache JSON files generated by backend provider.
- Useful for offline/cache mode or reproducible demos.

3. `local_data/support-data/Hardship Index of Chicago.csv`
- Hardship source file used by socioeconomic computations.

4. `local_data/frontend-public/report-figures/`
- Image assets used by homepage and story sections.

5. `api_export_summary.csv`
- Export status, response row counts, and source metadata per endpoint.

## Recommended Run Mode (local demo)
- Backend:
  - `DATA_SOURCE_MODE=cache`
  - `CACHE_DIR=data_cache`
- Frontend:
  - `NEXT_PUBLIC_API_BASE=http://127.0.0.1:8002/api`
  - `INTERNAL_API_BASE=http://127.0.0.1:8002/api`
